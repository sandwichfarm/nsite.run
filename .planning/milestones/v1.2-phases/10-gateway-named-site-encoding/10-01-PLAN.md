    GREEN phase: Create `packages/shared/src/base36.ts` with these two functions:

    `base36Encode(bytes: Uint8Array): string`
    - Convert 32 bytes to a BigInt (big-endian: bytes[0] is most significant)
    - Repeatedly divmod by 36n, collecting remainders
    - Map remainders to chars using alphabet "0123456789abcdefghijklmnopqrstuvwxyz"
    - Reverse the collected chars (divmod produces least-significant first)
    - Zero-pad to exactly 50 chars on the LEFT
    - Return the 50-char string

    `base36Decode(s: string): Uint8Array | null`
    - If s.length !== 50, return null
    - If s contains any char not in [a-z0-9], return null (use regex `/^[a-z0-9]{50}$/`)
    - Convert chars to a BigInt: for each char, multiply accumulator by 36n and add digit value
    - Convert BigInt to bytes: extract 32 bytes big-endian (shift right by 8 bits, 32 times)
    - If the BigInt requires more than 32 bytes (value >= 2^256), return null
    - Return the Uint8Array(32)

    Run tests — all must pass.

    REFACTOR: Clean up if needed. Ensure both functions are exported.
