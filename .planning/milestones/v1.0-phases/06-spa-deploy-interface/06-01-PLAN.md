Create the apps/spa/ directory with a complete Svelte 4 + Vite project:

1. **package.json** — name: "@nsite/spa", type: "module", private: true. Scripts: dev, build, preview, test (vitest --run). Dependencies:
   - svelte ^4.2.20, vite ^5.0.0, @sveltejs/vite-plugin-svelte ^3.0.0
   - tailwindcss ^3.4.0, postcss ^8.5.6, autoprefixer ^10.4.21
   - applesauce-signers ^5.1.0, applesauce-core ^5.1.0, applesauce-relay ^5.1.0
   - nostr-tools ^2.10.0
   - fflate ^0.8.2, nanotar ^0.1.1
   - qrcode ^1.5.4, @rollup/plugin-yaml ^4.1.2
   - devDependencies: vitest ^3.0.0, @types/qrcode ^1.5.0

2. **vite.config.js** — import svelte and yaml plugins. Configure for Svelte 4.
   ```js
   import { defineConfig } from 'vite';
   import { svelte } from '@sveltejs/vite-plugin-svelte';
   import yaml from '@rollup/plugin-yaml';
   export default defineConfig({ plugins: [svelte(), yaml()] });
   ```

3. **svelte.config.js** — minimal: `import { vitePreprocess } from '@sveltejs/vite-plugin-svelte'; export default { preprocess: vitePreprocess() };`

4. **tailwind.config.js** — content paths for `./src/**/*.{html,js,svelte,ts}`, extend theme for dark color palette (slate-900 background, purple/indigo accents).

5. **postcss.config.js** — tailwindcss + autoprefixer plugins.

6. **index.html** — Vite entry HTML: `<!DOCTYPE html>`, charset utf-8, viewport meta, title "nsite.run - Deploy to the Nostr Web", dark background color in body style, `<div id="app"></div>`, `<script type="module" src="/src/main.js"></script>`.

7. **src/main.js** — Mount App.svelte: `import App from './App.svelte'; import './app.css'; const app = new App({ target: document.getElementById('app') }); export default app;`

8. **src/app.css** — Tailwind directives: `@tailwind base; @tailwind components; @tailwind utilities;` plus base body styles: `body { @apply bg-slate-900 text-gray-100; }`

9. **src/App.svelte** — Placeholder root component: `<main class="min-h-screen"><h1 class="text-3xl font-bold text-center pt-20">nsite.run</h1><p class="text-center text-gray-400 mt-4">Deploy your site to the Nostr web</p></main>`

10. **vitest.config.js** — `import { defineConfig } from 'vitest/config'; export default defineConfig({ test: { environment: 'node' } });`

Do NOT add apps/spa to root deno.json workspace — this is an npm project.
