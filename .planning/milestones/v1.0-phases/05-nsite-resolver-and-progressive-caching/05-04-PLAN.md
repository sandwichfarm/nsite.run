    - Since `route()` is now async, ALL test assertions that call `route(...)` need `await`:
      - `const response = route(request)` becomes `const response = await route(request)`
    - The test functions need to be `async` (Deno test supports async test functions).
    - Resolver routing tests will need updating since handleResolver is now live. For router unit tests, the resolver will try to decode npub and hit the DB. Two options:
