    - extractNpubAndIdentifier("nsite.run:8080") returns null (root domain with port)

    - extractNpubAndIdentifier("bad chars!.npub1abc.nsite.run") returns null (invalid identifier characters)
    - extractNpubAndIdentifier("a.b.c.nsite.run") returns null (too many parts, no npub1 match at index 1)
