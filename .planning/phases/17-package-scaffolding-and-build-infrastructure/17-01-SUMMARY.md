## Decisions Made

- npm workspaces use `"*"` as the version specifier for workspace packages — the plan documented `workspace:*` which is pnpm/yarn syntax; npm resolves workspace packages by matching name in workspaces array using bare `*`
- `.npmrc` with `@jsr:registry=https://npm.jsr.io` is required because apps/spa depends on `@std/media-types` via `npm:@jsr/std__media-types` alias, and JSR packages live on npm.jsr.io, not registry.npmjs.org
