<action>
Create `packages/deployer/src/widget/NsiteDeployerElement.js` with a class that extends HTMLElement.

**Class structure:**

```javascript
import { STYLES } from './styles.js';
import DeployerWidget from '../components/DeployerWidget.svelte';

const FORWARDED_EVENTS = [
  'deploy-success', 'deploy-error', 'auth-change',
  'operation-start', 'operation-end', 'site-deleted'
];

export class NsiteDeployerElement extends HTMLElement {
  static observedAttributes = [
    'trigger-label', 'trigger-color', 'trigger-bg',
    'default-relay', 'default-blossom'
  ];
```

**Constructor:**
- Call `super()`
- `this._shadow = this.attachShadow({ mode: 'open' })`
- Initialize private fields: `this._svelteInstance = null`, `this._isOpen = false`, `this._extraRelays = []`, `this._extraBlossoms = []`, `this._signer = null`

**connectedCallback():**
- Call `this._renderTrigger()` to render the initial button

**disconnectedCallback():**
- If modal is open, call `this._closeModal()`
- If `this._svelteInstance`, call `this._svelteInstance.$destroy()` and set to null

**attributeChangedCallback(name, oldVal, newVal):**
- If `name === 'trigger-label'` and not open: re-render trigger button
- If `name === 'trigger-color'` or `name === 'trigger-bg'`: update CSS custom properties on shadow host
- If `name === 'default-relay'` or `name === 'default-blossom'` and `this._svelteInstance`: call `this._svelteInstance.$set({ [camelCase(name)]: newVal })`

**JS Properties (getter/setter pairs):**

```javascript
get extraRelays() { return this._extraRelays; }
set extraRelays(v) {
  this._extraRelays = Array.isArray(v) ? v : [];
  if (this._svelteInstance) {
    this._svelteInstance.$set({ extraRelays: this._extraRelays });
  }
}

get extraBlossoms() { return this._extraBlossoms; }
set extraBlossoms(v) {
  this._extraBlossoms = Array.isArray(v) ? v : [];
  if (this._svelteInstance) {
    this._svelteInstance.$set({ extraBlossoms: this._extraBlossoms });
  }
}

get signer() { return this._signer; }
set signer(v) {
  this._signer = v;
  if (this._svelteInstance) {
    this._svelteInstance.$set({ signer: v });
  }
}
```

**_renderTrigger():**
- Set `this._shadow.innerHTML` to:
  ```
  <style>${STYLES}</style>
  <button class="nsd-trigger">${this._escapeHtml(this._triggerLabel)}</button>
  ```
- Apply custom color/bg via CSS custom properties if attributes set
- Add click listener on the button to call `this._openModal()`

**get _triggerLabel():**
- `return this.getAttribute('trigger-label') || 'Deploy to nsite';`

**_openModal():**
- If already open, return
- Set `this._isOpen = true`
- Set `this._shadow.innerHTML` to:
  ```
  <style>${STYLES}</style>
  <button class="nsd-trigger" disabled>${this._escapeHtml(this._triggerLabel)}</button>
  <div class="nsd-overlay">
    <div class="nsd-modal">
      <button class="nsd-close">&times;</button>
      <div class="nsd-mount"></div>
    </div>
  </div>
  ```
- Add click listener on overlay for backdrop close (only if `e.target` has class `nsd-overlay`)
- Add click listener on close button
- Add keydown listener on document for Escape key
- Mount DeployerWidget into `.nsd-mount`:
  ```javascript
  const mountPoint = this._shadow.querySelector('.nsd-mount');
  this._svelteInstance = new DeployerWidget({
    target: mountPoint,
    props: {
      signer: this._signer,
      extraRelays: this._extraRelays,
      extraBlossoms: this._extraBlossoms,
      defaultRelay: this.getAttribute('default-relay') || undefined,
      defaultBlossom: this.getAttribute('default-blossom') || undefined,
    }
  });
  ```
- Set up event forwarding for all FORWARDED_EVENTS:
  ```javascript
  FORWARDED_EVENTS.forEach(eventName => {
    this._svelteInstance.$on(eventName, (e) => {
      this.dispatchEvent(new CustomEvent(eventName, {
        detail: e.detail,
        bubbles: true,
        composed: true
      }));
    });
  });
  ```

**_closeModal():**
- If not open, return
- Set `this._isOpen = false`
- If `this._svelteInstance`, call `$destroy()` and set to null
- Remove document keydown listener
- Call `this._renderTrigger()` to restore the button

**_onKeydown(e):**
- Bound method (stored reference for removal)
- If `e.key === 'Escape'`, call `this._closeModal()`

**_escapeHtml(s):**
- Replace `&`, `<`, `>`, `"` with HTML entities (same as stealthis pattern)
</action>
